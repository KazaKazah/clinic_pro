# Информационная система клиники

Первый рабочий модуль проекта на Django + Bootstrap 5: регистрация пациента, создание обращения, направление к врачу, создание приема и фиксация оплаты.

## Что уже реализовано

- вход пользователя в систему;
- панель регистратора;
- поиск пациента по ИИН, ФИО или телефону;
- регистрация нового пациента;
- справочники специальностей, врачей и услуг через Django Admin;
- создание обращения пациента;
- выбор врача, услуги, даты и времени приема;
- автоматическое подтягивание суммы из выбранной услуги;
- создание оплаты;
- карточка приема для врача;
- очередь пациентов для врача.

## Структура маршрута пациента

```text
Пациент пришел в клинику
-> поиск в базе
-> регистрация или выбор существующей карточки
-> создание обращения
-> выбор услуги и врача
-> создание приема
-> фиксация оплаты
-> карточка появляется у врача
```

## Запуск

```powershell
py -m pip install -r requirements.txt
py manage.py migrate
py manage.py createsuperuser
py manage.py runserver
```

После запуска:

- сайт: http://127.0.0.1:8000/
- админ-панель: http://127.0.0.1:8000/admin/

## Первичная настройка в админ-панели

1. Создайте специальности: терапевт, хирург, кардиолог и т.д.
2. Создайте пользователей для врачей.
3. Создайте врачей и привяжите их к пользователям.
4. Создайте услуги и укажите стоимость.
5. Зайдите как регистратор и создайте первое обращение пациента.

## Следующий этап

Логично продолжить модулем приема врача:

- начать прием;
- внести жалобы и анамнез;
- указать диагноз;
- назначить консультацию, обследование или госпитализацию;
- закрыть прием с заключением.
